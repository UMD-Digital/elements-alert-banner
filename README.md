# <umd-alert-banner> element

## Installation

```bash
yarn add @universityofmaryland/alertbanner
```

## Basic Usage

**Import the Alert Banner in your `js` if you bundle using a transpile with babel or typescript.**

```js
import '@universityofmaryland/alertbanner';
```

**HTML Usage**

```html
<umd-alert-banner>
  <!-- alert content -->
</umd-alert-banner>
```
